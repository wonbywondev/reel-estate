# 슬라이드 구성

1. **썸네일** — 주소 + 가격
2. **넓은 지도** — 동네 위치
3. **거리뷰** — 건물 주변 거리
4. **실내 사진** (옵션, 반복) — 각 사진마다 레이블 표시 (ex. 방1, 방2, 거실, 도면)
5. **지하철역 지도** — 역명 + 거리(m)
6. **근처 편의시설** (각 항목 한 화면)
   - 대형마트 / 다이소 / 백화점·아울렛 — 이름 + 거리
   - 편의점 — 이름 + 거리
   - 영화관 / 서점 — 이름 + 거리
   - 공원 — 이름 + 거리
7. **옵션 + 방향 + 준공연도 + 방 구성** — 옵션 목록, 방향 (ex. 남향), 준공연도, 방 구성 (ex. 방2 거실1 화장실1)
8. **가격 + 전세대출 정보**
9. **CTA**

---

## 메모

- 거리뷰: 건물 정면을 향한 뷰인지 확인 필요
- 실내 사진: 도면 이미지도 여기에 포함 가능
